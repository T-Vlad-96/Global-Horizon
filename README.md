
## Global Horizon 
A website where redactors can register their profiles and publish newspapers (articles) on any topic under their name.
 
Newspapers may have several topics and authors. Any newspaper can be edited or deleted at any time after publication.
Every author has own profile with personal information, contact data and experience info. 
Authors can delete their profiles or edit their personal information at any time.

The website has functionality that allows users to quick search for newspapers, redactors or topics they are insterested for.
Convenient navigation panel allows users quickly switch between sections of the website. 

## Technologies stack
Built with Python, Django Framework with help of Bootstrap5 and Crispy forms for better UX
####

## Installation & Setup
Clone down this repository. You will need PyCharm or other IDE that supports Django Template syntax + technology stack from requirements.txt.
